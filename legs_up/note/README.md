Legsup – Courte Échelle Réaliste (FiveM)
Description

Legsup est un script FiveM permettant à deux joueurs de coopérer pour réaliser une courte échelle réaliste, avec animations personnalisées, synchronisation réseau et physique en arc.

Fonctionnalités

Animations custom Blender (support / propulsion / saut)

Alignement automatique des joueurs

Physique réaliste avec trajectoire en arc

Vérifications Automatique:

Distance aux murs

Hauteur libre au-dessus

États du joueur (combat, chute, nage, etc.)

Cooldown configurable avec timer visuel

Anti-spam & anti-bug

Optimisé et léger (aucune dépendance 100% Standalone)

 Commandes

/legsup → Activer / désactiver le rôle de support

E → Se faire propulser si un support est disponible

/aforce → (debug) test de la force

/ragdoll → (debug)

 Configuration
LEGSUP_COOLDOWN = 5000
MIN_WALL_DISTANCE = 2.0
MIN_ROOF_HEIGHT = 3.0
ARC_UP_FORCE = 10.0
ARC_FORWARD_FORCE = 3.0
